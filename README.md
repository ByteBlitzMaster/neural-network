
# 🧠 Neural Network From Scratch (Python)


## 🚀 Overview

A fully functional **Neural Network built from scratch using Python and NumPy**, without relying on ML frameworks like TensorFlow or PyTorch.

This project focuses on understanding the **mathematics and algorithms behind deep learning**, including forward propagation, backpropagation, and optimization techniques.

---

## ✨ Features

* 🧮 Fully connected feedforward neural network
* 🔁 Forward propagation implementation
* 🔙 Backpropagation from scratch
* 📉 Gradient descent optimization
* 🔢 Custom activation functions (ReLU, Sigmoid, Tanh)
* 📊 Loss functions (MSE, Cross-Entropy)
* 🧠 Multi-layer architecture support
* 📈 Training visualization (loss vs epochs)

---


## 🛠️ Tech Stack

* **Language:** Python
* **Libraries:** NumPy, Matplotlib
* **Concepts:** Linear Algebra, Calculus, Optimization

---

## 📖 How It Works

1. Initialize weights and biases
2. Perform forward propagation through layers
3. Compute loss
4. Apply backpropagation using chain rule
5. Update parameters using gradient descent
6. Repeat for multiple epochs
